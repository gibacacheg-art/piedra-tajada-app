Manual de usuario

Guía breve para ventas, coordinación, responsables de área y administración. Este manual acompaña la operación diaria de reservas, eventos, cobros, documentos y seguimiento interno.

Descargar manual

1. Ingreso a la app

  1. Entra con tu correo y contraseña.
  2. Si tu acceso fue desactivado, un administrador debe volver a habilitar tu perfil.
  3. Tu contraseña se cambia desde Mi perfil.

2. Flujo principal del caso

  1. Nueva solicitud: ingresa cliente, fecha tentativa, horario, invitados y notas.
  2. Cotización: arma o revisa la propuesta comercial.
  3. Confirmar reserva: convierte la solicitud en evento operativo.
  4. Evento: desde aquí se coordinan responsables, tareas, cobros y documentos.
  5. Realizada / Cerrada: una vez ejecutado el evento, revisa cobros y pendientes antes de archivarlo.

3. Qué hace cada módulo

  • Reservas: puerta principal para crear, seguir y confirmar casos.
  • Calendario: vista de compromisos por fecha.
  • Clientes: ficha, historial y datos de contacto.
  • Cobros y facturación: cuotas, pagos, vencimientos y respaldo financiero.
  • Documentos: repositorio de archivos del caso.
  • Mi perfil: datos personales y cambio de contraseña.

4. Lectura rápida de estados

  • Nueva solicitud: caso recién ingresado.
  • Cotización enviada: propuesta comercial ya preparada.
  • Pre-reserva: caso casi listo para confirmarse.
  • Confirmada: el evento ya pasó a operación.
  • Realizada: el evento ocurrió.
  • Cerrada: ya no quedan pendientes operativos o financieros visibles.
  • Cancelada / Perdida: el caso salió del flujo activo.

5. Cobros y documentos

  • Registra cuotas o pagos desde el evento.
  • Usa los cobros registrados como referencia principal del saldo pendiente.
  • Sube comprobantes, facturas y respaldos documentales desde el evento o el módulo de documentos.
  • Si un archivo fue enviado por error a papelera, solo administración puede restaurarlo o eliminarlo definitivamente.

6. Roles de uso

  • Administrador general: acceso total, usuarios, papelera y ajustes internos.
  • Ventas: solicitudes, cotización, confirmación y seguimiento comercial.
  • Coordinador de evento: operación, responsables, tareas y control del evento.
  • Responsable de área: ejecución de tareas y seguimiento del trabajo asignado.

7. Buenas prácticas

  • Usa Confirmar reserva solo cuando el caso ya está listo para pasar a evento.
  • Archiva cuando quieras sacar algo del trabajo diario sin perder historial.
  • Usa papelera solo cuando fue un error y necesitas opción de recuperación.
  • No prometas un evento cerrado hasta revisar cobros y pendientes visibles.
